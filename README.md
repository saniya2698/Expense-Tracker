# 💰 Expense Tracker – Python File Handling Project

Expense Tracker is a console-based Python application designed to help users record, manage, and analyze their daily expenses efficiently. The project focuses on implementing file handling and data persistence using TXT files instead of databases, making it lightweight, beginner-friendly, and easy to understand.

The application allows users to add new expenses, view saved records, calculate total spending, generate category-wise expense summaries, and manage expense data through a simple menu-driven interface. All records are stored permanently in a text file, ensuring that data remains available even after the program is closed.

This project was developed to demonstrate core Python programming concepts such as functions, loops, conditional statements, file handling, exception handling, string manipulation, and data processing. It serves as a practical example of how real-world applications can store and retrieve data without relying on external databases.

## ✨ Key Features

* Add and store expense records permanently in TXT files
* View all saved expense records
* Calculate total expenses automatically
* Generate category-wise spending summaries
* Delete expense records when required
* Menu-driven user interface
* File-based data persistence
* Error handling for missing files and invalid operations
* Lightweight and beginner-friendly design
